# ようこそ、私のポートフォリオサイトへ！

このポートフォリオサイトはReactとViteで作っています。サイトのテーマは「和モダン」です。
和を表現するために、字体を明朝体、書道の墨汁・和紙をイメージした全体の色合い、浮世絵風の差し込み画像を採用しています。また、画面上部のナビゲーションバー、左側のスクロールバーで各項目へ遷移することも可能です。

## サイトの構成

### ホーム
・自己紹介
　名前、技術領域、現在のアルバイト、趣味、好きなものを書いています。

・アイコン
　実家の猫の写真を採用しています。ほとんどのSNSでこの写真をアイコンにしています。

### 経歴
・学歴
　高校卒業からこれまでの学歴を書いています。

・興味・関心
　私の興味や関心のある分野の浮世絵風の画像を表示しています。カーソルを合わせると文字が浮かび上がり、私の興味・関心のある分野が分かります。

・ハッカソン参加実績
　これまでのハッカソン参加実績を書いています。「詳細を見る」をクリックすると、参加したハッカソンのサイトへ遷移します。

### スキル
・技術スタック
　これまで経験のあるプログラミング言語、フレームワーク、ツールを紹介しています。

### プロジェクト
　これまでの個人開発、チーム開発のプロジェクトを紹介しています。「詳細を見る」をクリックすると、プロジェクトのサイトへ遷移します。

### お問い合わせ
　GitHub、Gmail、X、プレーリーカード（デジタル名刺）のリンクを張っています。
